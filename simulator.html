📖 Справка

MOV dst, src — загрузить значение

ADD / SUB dst, src — сложение/вычит.

MUL src — AX = AX × src

PUSH / POP reg — стек

CMP a, b — сравнить, флаги

JMP/JE/JNE/JL/JG label — переход

LOOP label — CX--, прыгнуть если ≠0

HLT — остановить выполнение
